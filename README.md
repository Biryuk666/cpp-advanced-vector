# Продвинутый вектор

## Описание
**Продвинутый вектор** - это структура данных на базе C++, которая является более совершенной версией [упрощенного вектора](https://github.com/Biryuk666/cpp-simple-vector). В данном контейнере произведена дополнительная работа с памятью и добавлен новый функционал, благодаря чему работать с ним стало удобнее и эффективнее. Однако, стоит учесть, что сам исполняемый файл будет занимать больше места в памяти, по сравнению с упрощенным аналогом. 

## Возможности
* Метод *Size()* возвращает размер контейнера.
* Метод *Capacity()* возвращает вместимость контейнера.
* Метод *Reserve(...)* изменяет вместимость контейнера, при условии, что новая вместимость больше, чем текущая.
* Метод *Resize(...)* изменяет размер контейнера. При увеличении размера новые элементы получают значение по умолчанию.
* Метод *PushBack(...)* работает как с *lvalue*, так и *rvalue* ссылками, и добавляет указанный элемент в конец контейнера.
* Метод *PopBack()* удаляет последний элемент контейнера.
* Метод *Insert(...)* работает как с *lvalue*, так и *rvalue* ссылками, и добавляет элемент в указанную позицию контейнера. Возвращаемым значением метода является итератор (указатель) на вставленный элемент.
* Метод *Erase(...)* удаляет элемент в указанной позиции контейнера и возвращает итератор (указатель) на следующий за удаленным элемент.
* Метод *Emplace(...)* позволяет производить более эффективную вставку элемента в указанную позицию и возвращает итератор (указатель) на него.
* Метод *EmplaceBack(...)* является частным случаем предыдущего метода и производит вставку в конец контейнера. Возвращаемым значением является ссылка на вставленный элемент.
* Метод *Swap(...)* обменивает значения с указанным контейнером.
* Метод *At(...)* возвращает ссылку на элемент с указанным индексом. Если в метод передать невалидный индекс, выбрасывает исключение *std::out_of_range*.
* Оператор "[ ]" возвращает ссылку на элемент с указанным индексом, но не выбрасывает исключение если индекс невалидный.
* Методы *begin()* и *cbegin()* возвращают итератор (указатель) и константный итератор на позицию первого элемента контейнера.
* Методы *end()* и *cend()* возвращают итератор (указатель) и константный итератор на следующую за последним элементом позицию. Данные итераторы запрещено разыменовывать, поскольку это приведет к неопределенному поведению.
* *Методы *rbegin()* и *crbegin()* возвращают обратный итератор (указатель) и константный обратный итератор на следующую за последним элементом позицию. Данные итераторы запрещено разыменовывать, поскольку это приведет к неопределенному поведению.
* Методы *rend()* и *crend()* возвращают обратный итератор (указатель) и константный обратный итератор на позицию первого элемента контейнера.

## Использование
Для использования **продвинутого вектора** необходимо скопировать файл *vector.h* в каталог с проектом и импортировать его как стороннюю библиотеку.
